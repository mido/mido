# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| < 1.3   | :x:                |

## Reporting a Vulnerability

[Create an issue](https://github.com/mido/mido/issues/new/choose) and select the appropriate feature, see below.

### Publicly

Select `Bug report`.

### Privately

Select `Report a vulnerability`.
